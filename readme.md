# Subsetsix

.NET CLI showing music scales on a guitar or bass fretboard. To run:

```
dotnet run show g dorian
```

Returns:

```
G Dorian [G, A, A#, C, D, E, F]

F ———◯———║———————│———◉———│———————│———◯———│———◯———│———————│———◯———│
C ———◯———║———————│———◯———│———————│———◯———│———◯———│———————│———◉———│
G ———◉———║———————│———◯———│———◯———│———————│———◯———│———————│———◯———│
C ———◯———║———————│———◯———│———————│———◯———│———◯———│———————│———◉———│
```

NOTE: Currently assumes a 4-string bass tuned drop-C.
